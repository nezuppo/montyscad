montyscad とは

montyscad は 3D CAD モデルを Python で作成し OpenSCAD スクリプトを生成するための Python パッケージです。

サンプルプログラム

montyscad パッケージを使ったサンプルプログラムです。

example.py
 1#!/usr/bin/env python3
 2
 3from decimal import Decimal
 4from montyscad import Scad
 5from montyscad import monty_symbols as ms
 6
 7scad = Scad()
 8
 9scad += [
10  '$fn=36;',
11  ms.union()(
12    ms.difference()(
13      ms.cube(size=30, center=True),
14      ms.sphere(r=Decimal('20.1'))
15    ),
16    ms.translate([0, 0, 30])(
17      ms.cylinder(40, r=10)
18    )
19  )
20]
21
22scad.write('/tmp/example.scad')

OpenSCAD スクリプトを生成

[インストール] を参考に montyscad パッケージがインストールされた状態で example.py を実行します。

$ python3 example.py

example.py の最後の行で scad.write('/tmp/example.scad') としているので、この python プログラムを実行した時に OpenSCAD スクリプト /tmp/example.scad が生成されます。

example.scad
 1$fn=36;
 2union() {
 3  difference() {
 4    cube(size=30, center=true);
 5    sphere(r=20.1);
 6  }
 7  translate([0, 0, 30]) {
 8    cylinder(40, r=10);
 9  }
10}

生成された example.scad を OpenSCAD で開き、作成した 3D CAD モデル の確認や STL ファイルへのエクスポート等を実施します。
